Instalasi Server

Panduan instalasi lengkap sistem CBT berbasis Laravel pada Ubuntu Server menggunakan Nginx, PHP 8.3, PostgreSQL, Supervisor, dan LiveKit Cloud untuk fitur video proctoring.

26 Langkah Instalasi — dari update sistem hingga optimasi produksi

Ubuntu 22.04 / 24.04 LTS
Web Server
Nginx + PHP 8.3-FPM
Database
PostgreSQL
Video Proctoring
LiveKit Cloud
Runtime
Node.js + FFmpeg
Daftar Isi
Langkah Instalasi
1
Update Ubuntu Server
Perbarui seluruh paket sistem dan atur zona waktu
# Perbarui paket sistem sudo apt update && sudo apt upgrade -y sudo timedatectl set-timezone Asia/Jakarta
2
Install Nginx
Web server untuk melayani aplikasi Laravel
sudo apt install nginx -y sudo systemctl enable nginx sudo systemctl start nginx sudo systemctl status nginx
3
Install PHP 8.3
Runtime PHP beserta seluruh ekstensi yang dibutuhkan Laravel
# Tambahkan repositori PPA ondrej/php sudo add-apt-repository ppa:ondrej/php -y sudo apt update # Install PHP 8.3 dan ekstensi sudo apt install -y \ php8.3 \ php8.3-fpm \ php8.3-cli \ php8.3-pgsql \ php8.3-mbstring \ php8.3-xml \ php8.3-curl \ php8.3-zip \ php8.3-bcmath \ php8.3-gd \ php8.3-intl \ php8.3-soap \ php8.3-imagick \ unzip \ curl \ git
4
Konfigurasi Upload Recording 1GB
Izinkan unggahan file besar untuk rekaman ujian

Edit file /etc/php/8.3/fpm/php.ini lalu cari dan ubah nilai-nilai berikut.

Buka File Konfigurasi
sudo nano /etc/php/8.3/fpm/php.ini
Cari Lalu Ubah
upload_max_filesize = 1024M post_max_size = 1024M memory_limit = 2048M max_execution_time = 3600 max_input_time = 3600 max_file_uploads = 100
Restart PHP-FPM
sudo systemctl restart php8.3-fpm
5
Install Composer
Package manager dependensi PHP untuk Laravel
curl -sS https://getcomposer.org/installer | php sudo mv composer.phar /usr/local/bin/composer composer --version
6
Install PostgreSQL
Database relasional yang digunakan PROCBT
sudo apt install postgresql postgresql-contrib -y sudo systemctl enable postgresql sudo systemctl start postgresql
7
Membuat Database PostgreSQL
Buat database dan pengguna khusus untuk aplikasi
# Masuk ke konsol PostgreSQL sebagai superuser sudo -u postgres psql # Buat database, user, dan berikan hak akses CREATE DATABASE procbt; CREATE USER procbt_user WITH PASSWORD 'passwordku'; GRANT ALL PRIVILEGES ON DATABASE procbt TO procbt_user; # Keluar dari konsol \q

Perhatian: Ganti 'passwordku' dengan kata sandi yang kuat.

8
Install Node.js
Runtime JavaScript untuk build asset frontend
# Tambahkan repositori Node.js 22.x curl -fsSL https://deb.nodesource.com/setup_22.x | sudo -E bash - sudo apt install nodejs -y node -v npm -v
9
Install FFmpeg
Library pemrosesan dan kompresi video rekaman ujian
sudo apt install ffmpeg -y ffmpeg -version
10
Install Supervisor
Process manager untuk menjaga worker queue tetap berjalan
sudo apt install supervisor -y sudo systemctl enable supervisor sudo systemctl start supervisor
11
Clone Repository PROCBT
Unduh source code aplikasi dari GitHub
cd /var/www/html git clone https://github.com/brehehe/cbt-new.git procbt cd procbt
12
Install Dependency Laravel
Install paket PHP via Composer dan build asset frontend
composer install npm install npm run build
13
Copy File Environment
Salin template konfigurasi lingkungan aplikasi
cp .env.example .env
14
Konfigurasi File .env
Sesuaikan variabel lingkungan dengan konfigurasi server Anda
APP_NAME="CBT Procbt" APP_ENV=production APP_KEY=base64:YOUR_APP_KEY APP_DEBUG=false APP_URL=https://domainanda.com APP_LOCALE=id APP_FALLBACK_LOCALE=id APP_FAKER_LOCALE=id_ID LOG_CHANNEL=stack LOG_STACK=single LOG_LEVEL=info DB_CONNECTION=pgsql DB_HOST=127.0.0.1 DB_PORT=5432 DB_DATABASE=nama_database DB_USERNAME=username_database DB_PASSWORD=password_database SESSION_DRIVER=database SESSION_LIFETIME=120 SESSION_ENCRYPT=false SESSION_PATH=/ SESSION_DOMAIN=null BROADCAST_CONNECTION=log FILESYSTEM_DISK=public QUEUE_CONNECTION=database CACHE_STORE=database MAIL_MAILER=log MAIL_FROM_ADDRESS="noreply@domainanda.com" MAIL_FROM_NAME="${APP_NAME}" VITE_APP_NAME="${APP_NAME}" TALLSTACKUI_PREFIX="ts-" APP_SLUG_NAME="login_universitas" IMPORT_QUESTION=true QUESTION_DUMMY=false QUESTION_PACKAGE=Lite SEB_ENABLED=true SEB_REQUIRE_ALL=false SEB_BROWSER_EXAM_KEY=YOUR_SEB_BROWSER_EXAM_KEY SEB_REQUIRE_BEK=false SEB_CONFIG_KEY=YOUR_SEB_CONFIG_KEY SEB_QUIT_PASSWORD=YOUR_SEB_PASSWORD CLEAR_SESSION_PASSWORD=procbt OCTANE_SERVER=frankenphp LIMIT_QUESTION_VIEW=false LIMIT_QUESTION_COUNT=5 LIVEKIT_URL=wss://your-livekit-domain.livekit.cloud LIVEKIT_API_KEY=YOUR_LIVEKIT_API_KEY LIVEKIT_API_SECRET=YOUR_LIVEKIT_API_SECRET LOGIN_IS_CREDENTIALS=true NEW_TEMPLATE=false
15
Generate APP Key
Buat kunci enkripsi unik untuk aplikasi
php artisan key:generate
16
Migrasi Database
Buat seluruh tabel dan isi data awal melalui seeder
php artisan migrate:fresh --seed
17
Login Default PROCBT
Kredensial bawaan setelah proses seeding selesai

Akun Administrator Default

Usernameprocbt
Password12345678

Segera ubah password setelah berhasil login pertama kali.

18
Permission Laravel
Atur hak akses direktori penyimpanan dan cache bootstrap
sudo chown -R www-data:www-data /var/www/html/procbt sudo chmod -R 775 storage sudo chmod -R 775 bootstrap/cache
19
Konfigurasi Supervisor Queue
Worker queue berjalan otomatis di background secara persisten
Buat File Konfigurasi
sudo nano /etc/supervisor/conf.d/procbt-worker.conf
Isi Konfigurasi
[program:procbt-worker] process_name=%(program_name)s_%(process_num)02d command=php /var/www/html/procbt/artisan queue:work database --sleep=3 --tries=3 --timeout=3600 autostart=true autorestart=true user=www-data numprocs=2 redirect_stderr=true stdout_logfile=/var/www/html/procbt/storage/logs/worker.log stopwaitsecs=3600
Reload Supervisor
sudo supervisorctl reread sudo supervisorctl update sudo supervisorctl start procbt-worker:*
20
Konfigurasi Nginx
Arahkan domain ke direktori publik Laravel
Buat File Konfigurasi
sudo nano /etc/nginx/sites-available/procbt
Isi Konfigurasi
server { listen 80; server_name domainanda.com www.domainanda.com; root /var/www/html/procbt/public; index index.php; client_max_body_size 1024M; proxy_read_timeout 3600; proxy_connect_timeout 3600; proxy_send_timeout 3600; fastcgi_read_timeout 3600; location / { try_files $uri $uri/ /index.php?$query_string; } location ~ \.php$ { include snippets/fastcgi-php.conf; fastcgi_pass unix:/run/php/php8.3-fpm.sock; } location ~ /\.(?!well-known).* { deny all; } }
Aktifkan Site & Restart Nginx
sudo ln -s /etc/nginx/sites-available/procbt /etc/nginx/sites-enabled/ sudo nginx -t sudo systemctl restart nginx
21
Install SSL HTTPS
Aktifkan enkripsi HTTPS menggunakan Certbot (Let's Encrypt)
sudo apt install certbot python3-certbot-nginx -y sudo certbot --nginx -d domainanda.com -d www.domainanda.com

Ganti domainanda.com dengan domain asli.

22
Setup LiveKit Cloud
Platform video real-time untuk fitur proctoring ujian
Langkah Pendaftaran
1
Buka website livekit.io, kemudian Register atau Login akun
2
Klik Create Project, lalu pilih Cloud Project
3
Masuk ke menu Settings pada proyek yang baru dibuat
4
Salin nilai: WebSocket URL, API Key, dan API Secret
23
Contoh Konfigurasi LiveKit
Format nilai LiveKit yang dimasukkan ke file .env
LIVEKIT_URL=wss://namaproject.livekit.cloud LIVEKIT_API_KEY=YOUR_API_KEY LIVEKIT_API_SECRET=YOUR_API_SECRET
24
Testing Queue Database
Pastikan antrian pekerjaan berjalan dengan benar
# Buat tabel queue (jika belum ada) php artisan queue:table php artisan migrate # Uji coba worker queue secara manual php artisan queue:work

Cache konfigurasi, route, dan view untuk performa optimal
